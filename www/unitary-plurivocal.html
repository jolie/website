The Unitary and Plurivocal Programming Language

UNITARY

A single language to manage the problems of digital integration in an holistic way

The unitary approach lets programmers specify a solution within a single language domain, without considering which technology will support the communication.

PLURIVOCAL

A tool able to dialogue with different applications using different protocols.

The property of being plurivocal allows the language to dialogue using different protocols and formats. In this way Jolie can adapt itself to the type of communication to be used according to the application which it interacts with.

UNITARY and PLURIVOCAL

A single means of problem solving that can dialogue with a multiplicity of existing realities

Being at once unitary and plurivocal makes Jolie, in its own way, unique and special. In fact, it allows both to design an integrated system in a uniform way, and to be able to dialogue with existing and future components.
