The IDE

We suggest to install the IDE "Atom":

Download Atom

Setup Atom for Jolie

Once installed Atom:

  1. Open Atom
  2. Click on "File" > "Settings"
  3. Select the left tab "Install"
  4. Search for "Jolie" Packages
  5. Install the package "language-jolie"

If you'd like to have also a linter (code and error highlighter), install also the package "linter-jolie".

Now you're ready to run your first Jolie program:
